🧠 الأسبوع 4: برنامج تسجيل دخول المستخدمين (Login System)

🎯 الهدف:
محاكاة نظام تسجيل مستخدمين مع إمكانية الدخول والتعامل مع البيانات.

✅ المهارات المستخدمة:
- OOP (Class, Object)
- File Handling (قراءة/كتابة من ملفات)
- Try/Except
- Dictionaries
- Loops + Logic

📋 الوظائف الأساسية:
1. تسجيل مستخدم جديد (اسم مستخدم وكلمة مرور)
2. حفظ البيانات في ملف
3. تسجيل الدخول
4. عرض رسالة ترحيب عند نجاح الدخول
5. التعامل مع أخطاء الإدخال أو بيانات خاطئة

🔥 تحديات إضافية:
- منع تكرار اسم المستخدم
- إخفاء كلمة المرور عند الإدخال (بـ getpass)
- تسجيل الوقت والتاريخ عند كل دخول ناجح
