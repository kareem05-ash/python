🧠 الأسبوع 3: برنامج آلة حاسبة متعددة الوظائف

🎯 الهدف:
إنشاء آلة حاسبة تتيح للمستخدم اختيار العملية الحسابية وتنفيذها باستخدام دوال منظمة.

✅ المهارات المستخدمة:
- Functions
- If / Match-case
- Loops
- Input Validation
- Math operations
- String formatting

📋 الوظائف الأساسية:
1. جمع / طرح / ضرب / قسمة رقمين
2. الجذر التربيعي
3. رفع رقم لقوة
4. إعادة البرنامج بعد كل عملية

🔥 تحديات إضافية:
- دعم العمليات على أكثر من رقم
- إدارة أخطاء القسمة على صفر
- تحويل النتائج من رقم عشري إلى أقرب عدد صحيح
